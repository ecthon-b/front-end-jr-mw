## 🧬 Executando em sua máquina

```terminal
# Clone este repositório
$ git clone https://github.com/ecthon/front-end-jr-mw.git

# Vá para a pasta do projeto e instale as dependências
$ yarn

# Após instalar as dependências, inicie o projeto
$ yarn start
```
